# NoteRx - 小红书笔记诊断技能

基于 874 条真实笔记 + 2465 条评论训练的多 Agent 诊断系统。

## 触发条件

用户发送以下内容时触发：
- "诊断笔记"
- "分析小红书"
- "优化笔记"
- "检查标题"
- "帮我看看这篇"
- 发送小红书截图或内容（需要分析）

## 核心能力

| 功能 | 说明 |
|------|------|
| 品类判断 | 根据内容判断美食/穿搭/科技/旅游/生活 |
| 标题诊断 | 评估标题结构（数字+情感+悬念） |
| 权重评分 | 基于品类差异化权重打分 |
| 钩子检查 | 评估钩子数量（3个最优，超过4个崩塌） |
| 评论预测 | 基于6种画像模拟真实评论区 |
| 优化改写 | 生成3个高分改写方案 |
| 黄金时段 | 17:00发布建议 |

## 使用流程

1. 用户发送笔记内容/标题
2. 判断内容品类
3. 分析标题结构、标签、内容
4. 基于品类权重评分
5. 生成优化建议 + 改写方案
6. 预测评论区反应

## 数据来源

- 874 条真实小红书笔记（美食183/穿搭278/科技235/旅游130/生活48）
- 2465 条评论分析
- Spearman 相关、线性回归、K-Means 聚类验证
