## 标题重写模板

### 模板1：数字 + 情感 + 悬念

```
原始：[泛泛标题]
改写：[数字] + [情感词] + [悬念/反转]

示例：
- "怎么省钱" → "5个省钱习惯，每月多存2000块"
- "护肤心得" → "30天逆袭！干皮变水光肌真实记录"
- "旅游攻略" → "这个小镇美到犯规！99%的人不知道"
```

### 模板2：经历型爆款

```
原始：[普通分享]
改写：有幸/曾经/第一次 + 强烈感受 + 具体描述

示例：
- "吃过一家店" → "有幸在亲戚家吃过一回，被惊艳到了！！"
- "用的产品" → "用了3年忍不住分享！真心吹爆这个..."
- "去过的地方" → "第一次去西藏，后劲太大了！"
```

### 模板3：实用数字型

```
原始：[经验分享]
改写：[数字] + [具体收益/效果] + [行动指引]

示例：
- "学习方法" → "从70分到95分！我只用这3个方法"
- "装修建议" → "20万装出50万效果！装修必看"
- "健身计划" → "每天10分钟，2周见效的减脂计划"
```

### 模板4：悬念反转型

```
原始：[陈述标题]
改写：千万别/除非/竟然 + 反转内容

示例：
- "好吃的店" → "千万别去这家店！去了就戒不掉"
- "推荐产品" → "用了这个，脸竟然小了一圈！"
- "旅行地点" → "这个海岛太绝了！除非去过才知道"
```

## 各品类改写策略

### 美食类

**核心**：极致口感 + 情感场景 + 实用价值

```
✅ 爆款结构：
"有幸在XX吃过一回，被惊艳到了！！"
"XX天吃了XX次！真的绝绝子"
"成本不到XX元，比店里的还好吃"
"千万别和室友一起看！会饿"

❌ 避免：
- 太普通的"分享一个好食谱"
- 纯步骤罗列无情感
- 营销感太强
```

### 穿搭类

**核心**：视觉 > 文字，但标题依然重要

```
✅ 爆款结构：
"这套绝了！被问链接问麻了"
"小个子显高神器！穿出大长腿"
"平价也能穿出高级感！💰百元搞定"
"一周穿搭不重样！懒人直接抄"

❌ 避免：
- 过度文字描述（文字只解释1.7%）
- 长文案标题
- 无视觉吸引的描述
```

### 旅游类

**核心**：标签策略 + 简单地名 + 情感表达

```
✅ 爆款结构：
"XX（地名）真的yyds！后劲太大了"
"在XX待了3天，不想回来了"
"XX小众目的地｜99%的人不知道"
"这个海边美到犯规！🏖️"

❌ 避免：
- 营销感标题（如"必去！绝美！")
- 过于复杂的地名堆砌
- 无情感的表达
```

### 科技类

**核心**：表情符号 + 感叹句 + 热点话题

```
✅ 爆款结构：
"终于找到了！🔥这个工具太牛了"
"不敢相信！XX竟然免费"
"救命！这个bug坑了我3天"
"XX功能太香了！后悔没早知道"

❌ 避免：
- 太正式的技术标题
- 缺乏情绪感染力
- 纯参数罗列
```

### 生活类

**核心**：个人觉醒 + 社会议题 + 实用指南

```
✅ 爆款结构：
"从XX到XX，我终于明白了..."
"XX岁，月薪XX万，这些话想告诉你"
"XX年，我放弃了XX，选择了..."
"真心建议：别XX（包括我）"

❌ 避免：
- 无共鸣的流水账
- 太负能量
- 纯炫耀无价值
```

## 钩子数量优化

| 钩子数 | 效果 | 说明 |
|--------|------|------|
| 0个 | ❌ | 无吸引力 |
| 1个 | ⚠️ | 偏弱 |
| 2个 | ✅ | 良好 |
| **3个** | ⭐ | **最优** |
| 4个+ | ❌ | 崩塌，用力过猛 |

**3个钩子示例**：
"救命！这个XX竟然只要99元，颜值还超高，千万别错过！"

## 内容优化检查清单

### 标题检查
- [ ] 有数字吗？（如：5分钟、3个方法）
- [ ] 有情感词吗？（如：绝绝子、被惊艳、yyds）
- [ ] 有悬念吗？（如：千万别、竟然、99%的人不知道）
- [ ] 字数在15-20字之间？
- [ ] 钩子数量 ≤ 3？

### 内容检查
- [ ] 有具体场景/经历吗？
- [ ] 有强烈感官描述吗？
- [ ] 有行动指引吗？
- [ ] 长度在品类最优区间？

### 标签检查
- [ ] 标签数在5-8个？
- [ ] 包含地点/话题标签？
- [ ] 无乱贴标签？

### 视觉检查
- [ ] 封面图清晰光线好？
- [ ] 主体突出无杂物？
- [ ] 有视觉吸引力？
