# 5维度评分标准

## 五维雷达评分

| 维度 | 说明 | 权重范围 |
|------|------|----------|
| 内容质量 | 标题、正文、标签的文案质量 | 0-100 |
| 视觉表现 | 封面图质量、构图、光线 | 0-100 |
| 增长策略 | 发布时间、钩子数量、互动引导 | 0-100 |
| 互动潜力 | 评论区激发能力、话题性 | 0-100 |
| 综合评分 | 加权总分 | 0-100 |

---

## 标题评分细则

### 结构评分（满分30）

| 结构 | 分数 | 示例 |
|------|------|------|
| 数字+情感+悬念全齐 | 25-30 | "5分钟搞定！这道菜我妈做了20年" |
| 数字+情感 | 18-24 | "3个动作，告别拜拜肉" |
| 情感+悬念 | 15-20 | "千万别点这道菜，除非..." |
| 数字+悬念 | 12-17 | "99%的人不知道这个技巧" |
| 只有数字 | 8-12 | "10个方法" |
| 普通陈述 | 3-7 | "分享一个好用的东西" |

## 视觉评分细则（基于用户描述）

### 封面质量（满分25）

| 描述 | 分数 | 说明 |
|------|------|------|
| 高质量，光线好，构图精致 | 22-25 | 专业级 |
| 质量不错，光线均匀 | 17-21 | 良好 |
| 画质一般，光线偏暗/过曝 | 10-16 | 需改进 |
| 模糊/杂乱/光线极差 | 1-9 | 致命问题 |

### 视觉风格（满分15）

| 风格 | 分数 | 说明 |
|------|------|------|
| 人物突出，画面干净 | 13-15 | 抓眼球 |
| 场景丰富，色彩协调 | 10-12 | 良好 |
| 背景杂乱/主体不清 | 4-9 | 干扰 |
| 纯文字/无视觉吸引力 | 1-3 | 无效 |

## 综合评分计算

```
综合评分 = 内容质量×品类权重["内容"] 
         + 视觉表现×品类权重["视觉"] 
         + 增长策略×品类权重["标题"]
         + 互动潜力×品类权重["互动"]
         + 标签×品类权重["标签"]
```

### 评分等级

| 分数 | 等级 | 说明 |
|------|------|------|
| 85-100 | S级爆款 | 有爆款潜力 |
| 70-84 | A级优质 | 质量不错，可优化 |
| 55-69 | B级普通 | 中等水平，需改进 |
| 40-54 | C级待优化 | 问题较多 |
| <40 | D级不合格 | 建议重写 |
