## 各品类评论特点

### 美食 — "共鸣场"

**特点**：
- 经验型评论多（36.9%）
- 常见"求地址"、"怎么做"
- 情感共鸣强，易引发讨论

**模拟评论示例**：
```
👍 种草型："这个配方绝了！已收藏"
💬 经验型："加一点料酒会更好吃，我试过"
🤔 质疑型："真的这么好吃？有点怀疑"
😂 调侃型："看完饿了，室友问我怎么了"
```

### 穿搭 — "种草场"

**特点**：
- 63% 正面评价
- 求链接评论多
- 种草型占比最高（25.4%）

**模拟评论示例**：
```
👍 种草型："太好看了吧！求链接求链接！"
🛒 求购型："多少钱？在哪里买？"
💬 经验型："我也有这件！搭牛仔裤绝了"
🤔 质疑型："身高多少？适合小个子吗"
```

### 旅游 — "向往场"

**特点**：
- 情感表达型评论多
- "好美"、"想去"类
- 标签相关讨论

**模拟评论示例**：
```
😍 种草型："美到犯规！五一就去这里"
❓ 求购型："酒店多少钱？求推荐"
💬 经验型："我去过！建议淡季去"
🤔 质疑型："滤镜很重吧？实际没那么美"
```

### 科技 — "辩论场"

**特点**：
- 27% 负面评价（最高）
- 质疑型评论多
- 技术讨论/争议

**模拟评论示例**：
```
🤔 质疑型："真的假的？感觉是软文"
💬 经验型："我用的感受是...其实一般"
👍 种草型："确实好用！已买"
😂 调侃型："又是智商税吧"
```

### 生活 — "共鸣场"

**特点**：
- 经验分享获赞是路人 6 倍
- 调侃型平均获赞 122（最高）
- 个人故事引发共鸣

**模拟评论示例**：
```
💬 经验型："同感！我也经历过..."
😂 调侃型："我直接住公司算了"
👍 种草型："写得太好了，感同身受"
🤔 质疑型："真的假的？这么容易？"
```

## 评论预测计算

### 基础预测

| 内容类型 | 预期评论类型 | 预期正面率 |
|----------|--------------|------------|
| 美食教程 | 经验型+种草型 | 65% |
| 穿搭分享 | 种草型+求购型 | 63% |
| 旅游打卡 | 种草型+求购型 | 70% |
| 科技测评 | 质疑型+经验型 | 45% |
| 生活观点 | 经验型+调侃型 | 58% |

### 爆款评论特征

- 有争议性话题 → 质疑型+辩论
- 具体经验分享 → 经验型高赞
- 情感共鸣强 → 调侃型高赞
- 求链接类 → 种草型但低赞

## 评论区运营建议

### 提升互动

| 策略 | 操作 | 效果 |
|------|------|------|
| 争议性话题 | 结尾问"你们觉得呢" | +30% 评论 |
| 求互动 | "你们遇到过吗" | +25% 评论 |
| 经验引导 | "欢迎分享你的..." | +20% 经验型 |
| 争议处理 | 及时回复质疑 | 提升信任 |

### 避免冷场

| 禁忌 | 说明 |
|------|------|
| 纯展示无问题 | 路人型占比高 |
| 敏感话题 | 可能引发负评 |
| 过时热点 | 互动低 |
